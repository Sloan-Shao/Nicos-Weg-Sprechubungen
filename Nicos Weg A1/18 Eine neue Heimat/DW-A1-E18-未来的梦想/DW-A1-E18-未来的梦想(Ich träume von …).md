## 对话练习
### 📋 练习说明

Lisa 和 Inge 对于她们的未来有什么样的愿望？请完成短文填空。

> **实用信息：** "Ich würde gerne …" 可以用于表达愿望。"Ich würde …" 则可以表达在特定情况下想做的事情。此时不需要 "gerne"。

## 📝 语法要点

### 1. `Ich würde gerne …` — 表达愿望

- **用法：** 用于表达一个愿望或想要做的事
- **结构：** `Ich würde gerne + 动词原形`
- **例句：**
  - Ich würde gerne reisen.（我想要旅行。）
  - Ich würde gerne wieder als Lehrerin arbeiten.（我想要再次当老师工作。）

### 2. `Ich würde …` — 表达在特定情况下想做的事

- **用法：** 表达在特定情况下想做的事情，**不需要** "gerne"
- **例句：**
  - Ich würde dann mit ihr schöne Ausflüge machen.（然后我会和她去美好的郊游。）

### 3. `Ich hätte gern …` — 想要拥有

- **用法：** 表达想要拥有某物
- **例句：**
  - Ich hätte gern einen eigenen Laden.（我想要有自己的店。）

### 4. `sich etwas wünschen` — 想要某物

- **用法：** 表达想要什么
- **例句：**
  - Ich wünsche mir so einen Laden.（我想要这样一个店。）
